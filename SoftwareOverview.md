# Software overview

The main repository is [`wheelsims`](https://github.com/WheelSims/wheelsims), and it contains the code related to the navigation in virtual environments, the graphical user interface to launch these environments, and the code to interact with external devices such as a D-Box incline platform.

The repository [`haptics`](https://github.com/WheelSims/haptics) contains Simulink Real-Time code to control simulators based on Speedgoat systems. It may in the future also contain Arduino code for simulators based on Arduino systems.

The documentation of each repository is found in their respective README.
